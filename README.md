# Cuis-Smalltalk-URI

Parse, serialize, and normalize
[RFC3986 URIs](https://datatracker.ietf.org/doc/html/rfc3986).
Also provides utilities for percent-encoding and query string parsing and
serialization.

Differs from the older
[Cuis-Smalltalk-URI-Kernel](https://github.com/Phantasus/Cuis-Smalltalk-URI-Kernel)
primarily in that it removes the dependency on OMeta2, but it is not intended
as a drop-in replacement.
